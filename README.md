# LinkedIn Profile Auditor

Free AI-powered LinkedIn audit tool. Covers all 14 profile sections with brutally honest feedback, a live score, and a 30-day action plan.

## Setup

1. Clone this repo
2. Run `npm install`
3. Create a `.env` file:
   ```
   ANTHROPIC_API_KEY=your_key_here
   ```
4. Run `npm run dev`

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Add `ANTHROPIC_API_KEY` in Vercel → Settings → Environment Variables
4. Deploy
